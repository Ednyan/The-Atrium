# Theme Assets

Place theme assets in this folder to customize the background environment.

## Supported Assets

### Ground Elements (`/ground`)
- Small decorative elements scattered across the world
- Recommended: stones, grass tufts, flowers, debris
- Format: PNG with transparency
- Size: 16x16 to 64x64 pixels recommended

### Particles (`/particles`)
- Animated particles that float across the screen
- Recommended: dust, sparkles, fireflies, snow
- Format: PNG with transparency
- Size: 8x8 to 32x32 pixels recommended

## File Naming
- Use descriptive names: `stone_1.png`, `grass_tuft.png`, `dust_particle.png`
- Files will be automatically loaded and distributed across the world
